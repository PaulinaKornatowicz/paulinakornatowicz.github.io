Study of concurrent image processing algorithms for a machine vision system

Master Thesis project developed during studies at Wrocław University of Technology (Advanced Applied Electronics).

The aim of the master thesis was to design and configure a multichannel machine vision system with the use of SoC (system-on-chip) device, implement the algorithms for detection of multiple objects on FPGA and perform the efficiency evaluation od the created system.

This was accomplished by using DE1-SoC Evaluation Board running Linaro Linux to acquire and analyze data from USB cameras connected to the board. The objects were detected by color and processed data was displayed on VGA screen. A web control panel was created for configuring the system.

The machine vision project created on DE1-SoC is a simple one and is still in development phase. It will be used as a base for more complicated designs for further analysis of image processing mechanisms using SoC technology.

  • Project duration: December 2015 - present
  • Category: Embedded Development, FPGA
  • Code: to be published on GitHub

Gear Fit Alarm

Android application created for personal use

An Android app that enables the user to set an alarm using Samsung Gear Fit smartwatch

The app uses Samsung SDK for Gear Fit to communicate with the watch. The user can set or cancel an already set alarm. The alarm itself is also handled by the app, a proper dialog is displayed on the watch enabling the user to snooze or disable an alarm. In case of Gear Fit not being paired at the time of alarm it can be dismissed automatically or handled by the user on the phone.

The app is published on Google Play Store.

  • Project duration: February 2017 - present
  • Category: Mobile Development

Smart Home app

Android application created for personal use

An Android app or controlling lights in apartment and in the future providing information about temperature both inside and outside the house.

The app uses Particle SDK to connect to a Particle Photon device that controls the leds. The hardware for control of the leds was designed using Eagle software. The user can turn the leds on and off using a button.

Further development:

- switch to RGB leds and let the user choose the color of the leds along with color transitions

- provide information about temperature using simple temperature sensors. Data should be collected using BLE (BLE connection in the app already tested, design of electronics in progress).

  • Project duration: May 2017 - present
  • Category: Mobile Development
