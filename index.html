An Android app for setting alarms to control lights (LEDs as well as all Philips Hue lights in the apartment) and temperature and in the future providing more automation. Gear Fit Alarm

mobile


An Android app that enables the user to set an alarm using Samsung Gear Fit smartwatch . The app uses Samsung SDK for Gear Fit to communicate with the watch. The user can set or cancel an already set alarm. The alarm itself is also handled by the app, a proper dialog is displayed on the watch enabling the user to snooze or disable an alarm. In case of Gear Fit not being paired at the time of alarm it can be dismissed automatically or handled by the user on the phone.

Study of concurrent image processing algorithms for a machine vision system

embedded develepoment, FPGA


Master Thesis project developed during studies at Wroc≈Çaw University of Technology (Advanced Applied Electronics). The aim of the master thesis was to design and configure a multichannel machine vision system with the use of SoC (system-on-chip) device, implement the algorithms for detection of multiple objects on FPGA and perform the efficiency evaluation od the created system. This was accomplished by using DE1-SoC Evaluation Board running Linaro Linux to acquire and analyze data from USB cameras connected to the board. The objects were detected by color and processed data was displayed on VGA screen. A web control panel was created for configuring the system.
