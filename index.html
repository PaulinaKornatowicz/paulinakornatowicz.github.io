Side projects.

Snooker Tracker


An Android app for tracking snooker matches. Choose your favorite snooker player and be notified about their upcoming matches as well as live scores. App uses data from rest api, although as a practice to learn GraphQL I plan to create my own API. Written completely in Kotlin using the newest architecture components from Google, kotlin coroutines and following clean architecture patterns.

Smart home


An Android app for setting alarms to control lights (LEDs as well as all Philips Hue lights in the apartment) and temperature and in the future providing more automation. The app connects with Raspberry Pi acting as REST API to send information when to change temperature and set lights. The hardware for control of the leds was designed using Eagle software and is controlled by STM Nucleo microcontroller.

Study of concurrent image processing algorithms for a machine vision system


Master Thesis project developed during studies at Wrocław University of Technology (Advanced Applied Electronics). The aim of the master thesis was to design and configure a multichannel machine vision system with the use of SoC (system-on-chip) device, implement the algorithms for detection of multiple objects on FPGA and perform the efficiency evaluation od the created system. This was accomplished by using DE1-SoC Evaluation Board running Linaro Linux to acquire and analyze data from USB cameras connected to the board. The objects were detected by color and processed data was displayed on VGA screen. A web control panel was created for configuring the system.
