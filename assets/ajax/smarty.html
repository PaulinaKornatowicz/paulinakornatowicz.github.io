Smart Home app

An Android app for setting alarms to control lights (LEDs as well as all Philips Hue lights in the apartment) and temperature and in the future providing more automation.

The app connects with Raspberry Pi acting as REST API to send information when to change temperature and set lights. The hardware for control of the leds was designed using Eagle software and is controlled by STM Nucleo microcontroller.

Further development:

- apart from setting alarms, user should be able to turn all light on and off and set tempertaure for each room separately
